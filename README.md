# Curve's CRV Airdrop

The CRV airdrop occurred on the 13th of August, 2020.

A smart contract was setup for claiming and users within the airdrop had their claims vest over a 1 year period.  The claim smart contract is: 0x575CCD8e2D300e2377B43478339E364000318E2c

This repo includes a simple CSV file with all the addresses and the amounts they were allocated.
